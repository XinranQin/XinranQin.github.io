# Xinran Qin Personal Homepage

This folder contains the static files for a GitHub Pages personal website.

## Files

- `index.html`: homepage
- `styles.css`: page styles
- `qxr.jpg`: profile photo
- `resume-QXR.pdf`: CV

## Deploy to GitHub Pages

Create a GitHub repository named:

```text
XinranQin.github.io
```

Then run:

```bash
git init
git add .
git commit -m "Create personal homepage"
git branch -M main
git remote add origin https://github.com/XinranQin/XinranQin.github.io.git
git push -u origin main
```

The website will be available at:

```text
https://XinranQin.github.io
```
